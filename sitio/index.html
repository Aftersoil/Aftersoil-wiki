# 已完成

  • 渐进式增强WEB应用，pwa。
  • 响应式设计，适配多种终端。

# 未完成

  • 图片懒加载
  • 内置多语言
  • 支持多款评论插件
  • 内置文章搜索
  • 自定义静态资源 CDN
  • 文章版权信息
  • 自定义代码高亮方案
  • 打赏功能

  • live2d 看板娘
  • 暗色模式
