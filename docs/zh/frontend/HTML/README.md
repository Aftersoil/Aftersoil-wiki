## web运行原理
web：运行在internet上的B/S结构的应用程序，俗称网站internet：为web运行提供了网络环境<br>
web的工作原理：基于浏览器和服务器以及通信协议来实现的数据的传输和展示

## 通信协议
规范了数据如何打包和传递
