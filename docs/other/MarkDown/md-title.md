# Markdown 标题

Markdown 标题有两种格式。

## 使用 = 和 - 标记一级和二级标题

= 和 - 标记语法格式如下：

```md
我展示的是一级标题
=================

我展示的是二级标题
-----------------
```

显示效果如下图：

![alt text](https://images.aftersoil.xyz/wiki/image/MarkDowm/md-title-1.png)

## 使用 # 号标记

使用 `#` 号可表示 1-6 级标题，一级标题对应一个 `#` 号，二级标题对应两个 `#` 号，以此类推。

```md
#       一级标题
##      二级标题
###     三级标题
####    四级标题
#####   五级标题
######  六级标题
```
